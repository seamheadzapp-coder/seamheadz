# Seamheadz Contract Tracker

Fantasy baseball salary cap & contract management for the Seamheadz league.

## Setup

1. Clone this repository
2. Run `npm install`
3. Copy `.env.local.example` to `.env.local` and fill in your Supabase anon key
4. Run `npm run dev` to start locally

## Deploy

Connected to Vercel — push to main branch to deploy automatically.

## Tech Stack

- Next.js 14
- Supabase (database)
- Tailwind CSS
- Deployed on Vercel
